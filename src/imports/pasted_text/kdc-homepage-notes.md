KDC HOMEPAGE REFINEMENT NOTES — BASED ON CURRENT MOCKUPS

GENERAL CREATIVE DIRECTION

The homepage is moving in the correct direction visually. The goal now is refinement, hierarchy, editorial pacing, and luxury restraint.

The website should feel:
- editorial
- cinematic
- fashion-world
- emotionally branded
- calm and elevated
- like a luxury creative consultancy

NOT:
- startup
- tech agency
- marketing template
- portfolio-heavy
- influencer/personal brand

The current direction already feels aligned with:
- Toteme
- Kallmeyer
- Another Tomorrow
- LouLou de Saison
- fashion campaign microsites

The next phase is refinement and intentionality.

--------------------------------------------------
HEADER / NAVIGATION
--------------------------------------------------

CURRENT STATUS:
Strong overall structure and spacing.

CHANGES:
- Increase navigation opacity slightly.
- Navigation text is currently too faint/light.
- Use #5E5954 for navigation instead of lighter gray.
- Increase spacing between nav items by approximately 10–15%.
- Lower nav vertically by a few pixels for better visual balance.

HEADER SHOULD FEEL:
- quiet
- restrained
- editorial
- calm
- expensive

Avoid:
- overly bold nav
- sharp contrast
- overly visible UI feeling

--------------------------------------------------
HERO SECTION
--------------------------------------------------

CURRENT STATUS:
This is now the strongest part of the site.

The fade-in animation works extremely well and should remain.

The current cropped Paris image feels:
- cinematic
- editorial
- fashion-oriented
- campaign-like
- emotionally branded

The asymmetrical crop should remain.

DO NOT:
- center the subject
- sharpen the image heavily
- make the image overly crisp
- make it feel influencer-style

The softness + motion is what makes it feel luxury/editorial.

--------------------------------------------------
HERO IMAGE ADJUSTMENTS
--------------------------------------------------

- Reduce blur slightly by approximately 10%.
- Keep cinematic softness.
- Preserve texture and environmental detail.
- Add extremely subtle film grain/noise.
- Maintain atmospheric softness.

The image should feel:
campaign-driven rather than portrait-driven.

- cinematic
- calm
- premium
- emotionally restrained

NOT:
- loud
- bold advertising
- startup hero copy

HERO SUBTITLE
--------------------------------------------------

CURRENT ISSUE:
Subtitle competes too much visually.

CHANGE:
- Reduce font size by ~15–20%.
- Increase tracking slightly.
- Reduce opacity slightly.
- Create more separation from headline.

The subtitle should feel almost whisper-like.

--------------------------------------------------
SECTION: WHO WE WORK WITH
--------------------------------------------------

CURRENT STATUS:
Very strong overall composition.

This section feels closest to true luxury editorial design.

REFINEMENTS:
- Reduce headline size slightly (~5–8%).
- Increase vertical spacing between:
  - section label
  - headline
  - body copy
- Slightly darken body copy for readability.
- Increase paragraph width slightly.

The section should breathe more.

CURRENT FEELING:
Slightly compressed vertically.

GOAL:
More calm editorial whitespace.

--------------------------------------------------
SECTION: THE PROBLEM
--------------------------------------------------

CURRENT STATUS:
Strong copy and positioning.

CURRENT ISSUE:
Layout still feels too centered and balanced.

REFINEMENTS:
- Push content slightly further left.
- Increase asymmetry.
- Narrow body copy slightly.
- Increase body copy line-height.
- Add more whitespace underneath section.

The section should feel:
editorially composed rather than template-centered.

--------------------------------------------------
SECTION: VALUE / BENEFITS
--------------------------------------------------

CURRENT ISSUE:
This section currently feels the weakest structurally.

The repeated stacked statements feel:
- too repetitive
- too document-like
- too evenly spaced

NEEDED CHANGES:
- Convert into more editorial grid structure.
- Introduce alternating rhythm/alignment.
- Reduce vertical spacing between value statements.
- Create grouping and hierarchy.

Possible direction:
- Left column = small label/introduction
- Right column = larger statements
OR
- staggered alternating statements

The goal is:
editorial pacing and rhythm variation.

--------------------------------------------------
OVERALL PAGE LAYOUT
--------------------------------------------------

CURRENT ISSUE:
Too many sections use similar centered alignment and rhythm.

NEEDED:
- more asymmetry
- more editorial tension
- more whitespace imbalance
- more visual rhythm variation

Luxury websites rely on:
restraint + pacing + composition.

Avoid:
- perfect symmetry
- equal spacing everywhere
- obvious template structure

--------------------------------------------------
TEXTURE / ATMOSPHERE
--------------------------------------------------

Add extremely subtle:
- paper texture
- film grain
- organic noise

Especially in:
- ivory background sections
- whitespace areas

The site should feel tactile and atmospheric.

Avoid:
- flat sterile backgrounds
- glossy UI feeling

--------------------------------------------------
MOTION / INTERACTION
--------------------------------------------------

Current fade-in animation is working extremely well.

Continue this motion direction throughout the site.

Recommended motion:
- soft fade-up text
- slow image reveals
- subtle opacity transitions
- restrained hover states
- cinematic pacing

Avoid:
- bouncy motion
- aggressive transitions
- startup-style animations

Motion should feel:
quiet, cinematic, restrained.

--------------------------------------------------
OVERALL POSITIONING GOAL
--------------------------------------------------

The site should feel like:
a luxury fashion-world strategic consultancy.

NOT:
a freelancer portfolio or marketing agency.

The homepage should communicate:
- editorial intelligence
- campaign thinking
- luxury brand fluency
- creative direction sensibility
- CRM + storytelling expertise
- fashion/lifestyle positioning

The emotional goal:
Visitors should feel like they entered a calm, elevated, editorial brand world — not a typical agency website.
